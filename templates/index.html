What the hell is this?

Imaginative Go is a website that created with Go lang. It contains many go lang working samples that for web and non-web application. It demonstrate what Go lang can achieve. You can contribute!

I believe that the best way to learn new programming language is by directly implement the knowledge into a project.
-- Sony Arianto Kurniawan, Imaginative Go Project Maintainer

This imaginative (but super real) project, we implement our knowledge during learning Go and we want to share with the community. We don't use any framework and forgive us if the code still not efficient or optimal, since this project will just give you an idea how to achieve something in Go language.

So let's start with code samples.
