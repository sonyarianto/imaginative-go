Hi, I'm Imaginative Go!

Imaginative Go is a website that created with Go. It contains many Go working samples code that useful for web and non-web application. It mainly demonstrate what Go can achieve. You can contribute!

This imaginative (but super real) project, we implement our knowledge during learning Go and we want to share with the community. We don't use any framework and forgive us if the code still not efficient or optimal, since this project will just give you an idea how to achieve something in Go language.

NOTES:
You can see this web code at file imaginative-go.go.
You can fork this website on your local machine and run docker-compose up -d then go to your browser at http://localhost:9899.
